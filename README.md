Projeto Final do CC50
Com Grande orgulho que eu apresento esse projeto 
Um projeto que levou um tempo para ser concluido mas finalmente está aqui
pode parece um gerenciador de senhas simples mas foi feito com muito esforço e dedicação 
para poder adquirir o meu certificado de ciẽncia da computação por harvard
Bom sem mais delongas 
Fiz o projeto em python por ser mais simples e ser um dos requisitos do projeto final 
eu mesclei SQL e criptografia para pode ajudar 
o SQL eu optei por usar o lite já que os execicios que eu havia feeito foram por ele 
e tambem pela praticidade de não precisar configurar tanto 
Ao executar o gerenciador vocẽ se depara com 5 opcões
sendo elas:
[1] Adicionar nova senha
[2] Ver senhas salvas
[3] Buscar senha
[4] Excluir senha
[5] Sair
para poder haver um manejo melhor das opções que voce queira fazer
o meu intuito foi fazer algo mais simples com que eu havia aprendido ao decorrer do curso 
Sendo que gostaria de implementar tambem a minha prática em ciber segurança
